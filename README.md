## 🌀 Descripción

**Nagaro** es una página web de presentación de **animes doblados y subtitulados**, pensada para los amantes del anime que buscan un lugar moderno, elegante y rápido para explorar sus series favoritas.  
El objetivo es ofrecer una interfaz fluida, con **efectos visuales responsivos**, y mantener una experiencia limpia sin distracciones.

---

## 🌈 Características principales

- 🎬 Catálogo de animes doblados y subtitulados  
- 🔍 Buscador interactivo  
- 📱 Diseño totalmente **responsivo**  
- 💫 Fondos con **degradados y transparencias** para un estilo moderno  
- 💾 Compatible con hosting estático (GitHub Pages, Vercel, etc.)  
- 🧩 Código modular y fácil de editar
